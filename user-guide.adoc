== Обзор системы
Вы используете версию API: {product-version}. Данная документация описывает методы интеграции.

== Установка
Для работы с библиотекой в проекте на C#, выполните:

[source,csharp]
----
// Инициализация сервиса
public void ConfigureServices(IServiceCollection services)
{
    services.AddLibraryApi(options => {
        options.Version = "{product-version}";
    });
}
----
